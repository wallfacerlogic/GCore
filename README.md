# GCore  
#### A simple RISC CPU on Lichee Nano FPGA  
##### Version:0.3.6 Beta  

## OP list  

- NOP   0000  
- JUMP  0001  
- SAVE  0010  
- LOAD  0011  
- LOADI 0100  
- SLL   0101  
- ADD   1000  
- SUB   1001  
- AND   1010  
- OR    1011  
- XOR   1100  
- SLT   1110  
- BZ    1111
